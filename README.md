# [ Installation ]
```
$ pkg update upgrade
$ pkg install git python2
$ git clone https://github.com/GILILLOMX/hackface
$ cd hackface
```

# [ Setup ]
```
$ pip2 install -r requirements.txt
```
# [ Running ]
```
$ python2 hackface.py
```
